# Marcus Astro Theme

This repository is built and deployed as an Astro site via GitHub Actions.

- Production URL: <https://sudheer.github.io/marcus-astro/>
- Preview URL: <https://sudheer.github.io/marcus-astro/preview/>

If you are seeing this page, GitHub Pages is currently configured to build from `docs/`.

## Recommended setting

Use the workflow-based branch deployment:

- **Settings → Pages**
- **Source**: Deploy from a branch
- **Branch**: `gh-pages`
- **Folder**: `/(root)`
